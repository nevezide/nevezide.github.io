A Blog
Partie 1 : Layout
Partie 2 : Sections
Partie 3 : Articles


L'objectif de ce mini projet est de réaliser un site web de type blog typique.

IMPORTANT :
  • On considère que les tags : section, article, main se comportent comme des div (qu'on peut d'ailleurs utiliser)
  • Les parties sont à réaliser dans l'ordre car chaque partie est un ajout à la précédente.
  • Hormis pour la flèche de retour en haut de la page, on n'utlise aucun positionnement CSS.
BONUS :
  • Une variante de cet exercice peut être de le refaire avec les positionnement CSS.
  • Jouer avec différentes structures de layout, en reproduisant des templates existants ici : Theme forest WordPress



Partie 1 : Créer le layout du site

  • Un entête et un pied de page
  • Une flèche pour remonter en haut du site
TIP : Mettre du texte dans le conteneur principal pour voir le comportement des éléments

Partie 2 : Ajout des sections

  • Les sections permettent de découper le conteneur principal afin de structurer les contenus de la page.
Exemple : 3 colonnes, 3 lignes, une mozaïque

Partie 3 : Ajout des articles

  • Le plus petit élement dans un blog, il va contenir le contenu (texte et images) du blog.
  • Les articles se positionnent à l'intérieur des sections
